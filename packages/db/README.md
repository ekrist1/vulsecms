# @vulse/db

libsql adapter, migration runner, and core schema for the
[Vulse](https://github.com/ekrist1/vulsecms) CMS.

```sh
pnpm add @vulse/db
```

Typically consumed via `@vulse/host`'s `prepareDatabase` helper. See
[docs/database.md](https://github.com/ekrist1/vulsecms/blob/main/docs/database.md).
