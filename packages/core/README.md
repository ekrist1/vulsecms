# @vulse/core

Blueprints, content service, HTTP API, event bus, mailer, and module loader
for the [Vulse](https://github.com/ekrist1/vulsecms) CMS.

## Install

```sh
pnpm add @vulse/core
```

This package is normally consumed indirectly through `@vulse/host`. See
the [architecture guide](https://github.com/ekrist1/vulsecms/blob/main/docs/architecture.md)
for the full extension model and the [upgrade guide](https://github.com/ekrist1/vulsecms/blob/main/docs/upgrading.md)
for the version-to-version migration story.
