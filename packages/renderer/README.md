# @vulse/renderer

Vue 3 field renderers used by Vulse blueprints. Shared by the `@vulse/admin`
SPA and by any frontend that wants to render Vulse block content (e.g. an
Astro site fetching content via `@vulse/astro`).

```sh
pnpm add @vulse/renderer
```

Ships a `./styles` export for the renderer CSS. Imported automatically by
`@vulse/admin`.
