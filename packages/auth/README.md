# @vulse/auth

Authentication, users, groups, and RBAC for [Vulse](https://github.com/ekrist1/vulsecms).
Wraps `better-auth` and exposes admin routes for user/group management.

```sh
pnpm add @vulse/auth
```

Typically consumed via `@vulse/host`'s `createDefaultAuth` helper. See
[docs/auth.md](https://github.com/ekrist1/vulsecms/blob/main/docs/auth.md).
